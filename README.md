# Septic Inspection Services Website

Professional website for septic tank inspection services. This website provides information about professional septic system evaluations, maintenance recommendations, and inspection scheduling.

## Features

- Professional septic tank inspection information
- SEO optimized content
- Mobile-responsive design
- Fast loading performance

## Technology Stack

- HTML5
- CSS3
- JavaScript
- Vite (Build tool)

## Development

To run this project locally:

1. Clone the repository
```bash
git clone https://github.com/BetoWH13/septicinspection.git
cd septicinspection
```

2. Make your changes
3. Commit and push your changes
```bash
git add .
git commit -m "Your commit message"
git push
```

## Deployment

This website is deployed on Netlify at [septicinspection.xyz](https://septicinspection.xyz)
